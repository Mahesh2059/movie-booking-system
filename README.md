# Movie Ticket Booking System

## Setup Instructions

1. Clone the repo
2. Import database:
   - Open phpMyAdmin
   - Create database: movie_ticket_booking
   - Import movie_ticket_booking.sql

3. Update database connection in conn.php:
   - host: localhost
   - user: root
   - password: your_password

4. Run project on localhost
